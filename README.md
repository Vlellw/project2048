# Project Goal:
### Creation of the game "2048" based on the Python library pygame. In our game, the user will be able to move tiles in four directions (left, right, up, down), merging them when they have the same numerical value. As a result of merging, the value of the tile increases. If tiles fill the entire game board and cannot be moved, the user loses. However, if the user reaches a tile with the value 2048, they win. We want to implement our project as an interesting and engaging, visually beautiful game. Our goal is also to create various difficulty levels.

## Difficulty Levels:
### We want to create 2 difficulty levels. 
On the first level (Classic game), the user will play our ‘2048’ game until they reach a tile with the value 2048. 
On the second level (Hard game), the user can merge tiles indefinitely — there is no numerical limit like in the first level.

## Future Development Prospects:
### In the future, we believe it is necessary to increase the number of levels. For example, create a level with a limited time for making decisions, or implement a level with ‘obstacles’ inside the game field. 
It is also worth adding the possibility of online multiplayer, where users could play against each other in real time.
